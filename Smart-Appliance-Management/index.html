PROJECT FEATURES

ESP8266 MODULE

Person counter - Calculates the number of people inside a room. Number of persons inside the room is important factor in managing the appliances efficiently.

State monitoring - Monitor and records the states of devices whether the device is on or off. Knowing the states of devices is necessary to control devices accordingly.

FIREBASE MODULE

Remote Monitoring - Cloud technology is used to store the device states and person count inside a room. Cloud eliminates the drawback of range of monitoring as the devices can be controlled anywhere and can be monitored from anywhere.

FLUTTER MODULE

User application - An user application that allows the user to register the devices present inside the room for remote monitoring. The application alerts the user whenever the appliances are unnecessarily used. User can remotely control the devices using the application.

WEBSITE MODULE

Website manual - A website manual that provides complete guide about the usage of the system. User also get to know the amount of units consumed and electricity tariff per day from which they can efficiently manage their devices.
